# UIApplicationsObjC

An basic UI Applications to use on a Objective C project.

## Requirements

```
iOS 13+
Xcode 11
Objective C
UIKit
```

## This class uses:

```
UITabBar
UISlider
UISwitch
UISegmentedControl
UIPageControl
UIPickerView
UIDatePickerView
```
